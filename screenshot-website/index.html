Installation

1. Clone the repository

git clone https://github.com/yagupta77/screenshot-analysis.git
cd screenshot-analysis

2. Install Python dependencies

pip install -r requirements.txt

3. Install Tesseract OCR

Tesseract OCR is required for image text extraction:

Download and install from UB Mannheim's Tesseract installer

sudo apt update
sudo apt install tesseract-ocr
brew install tesseract
Requirements
  • Python 3.6+ Required
  • OpenCV (cv2) Required
  • Pytesseract Required
  • Tesseract OCR Required
  • Python-dotenv Required

Usage

Analyzing an Image

python analyze_screenshot.py path/to/your/image.png

This will extract text from the image and analyze it for security issues.

Analyzing a Text File

python analyze_screenshot.py --text path/to/your/file.txt

This will analyze the text content of the file for security issues.

Interactive Text Mode

python analyze_screenshot.py

Enter text to analyze. Press Ctrl+D (Unix) or Ctrl+Z (Windows) followed by Enter when finished.

Getting Help

python analyze_screenshot.py --help

Display usage instructions and supported options.
